# Переименновывает окна Винды

<ol>
  <li>Вводится НазваниеПроцесса.exe</li>
  <li>Нажимается кнопка "Поиск"</li>
  <li>Отображается текущее имя приложения</li>
  <li>Вводится новое имя окна</li>
  <li>Нажимается кнопка "Переименновать"</li>
</ol>
